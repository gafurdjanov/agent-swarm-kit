# ILogger

ILogger interface for logging messages.

## Methods

### log

```ts
log: (topic: string, ...args: any[]) => void
```

Logs a message.

### debug

```ts
debug: (topic: string, ...args: any[]) => void
```

Logs a debug message.

### info

```ts
info: (topic: string, ...args: any[]) => void
```

Logs a info message.
